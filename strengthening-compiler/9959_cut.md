## Postconditions

```zig

assert(index + needle.len <= haystack.len);
assert(before.len + needle.len + after.len == haystack.len);
```

**cutScalar** on match:

```zig

assert(before.len + 1 + after.len == haystack.len);
```

- Sequence and scalar splits match upstream cases
